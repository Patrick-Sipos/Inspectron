A desktop application designed for the automatic detection of components and defects on printed circuit boards (PCBs) using artificial intelligence. Built with a YOLOv11-Large model for object detection and a PyQt5 graphical interface, the app allows users to load PCB images, process them locally, and visualize detection results.
